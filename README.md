## koa-spring-client
一个标准的中后台系统，一个NodeJs服务的试金石。

## 使用步骤
1. npm install // 安装依赖

2. npm start // 开启本地在线调试

3. 浏览器输入 http://localhost:8905/

4. 登陆页  
账号：dom  
密码：123456

5. 点击菜单规则列表，体会增删查改
